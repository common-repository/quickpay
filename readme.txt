== Description ==

== Frequently Asked Questions ==

= How do I use the plugin? =

To use the plugin, Simply create an account on <a href="https://quickpay.bluecube.co.ug">Quickpay</a>

Visit developer settings on the menu and create the api credentials. Copy the Api Key and save it under the quickpay plugin settings on the WordPress admin panel.

Simply insert the above shortcode on any page and the Quickpay form will be displayed.
